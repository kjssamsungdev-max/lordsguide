Display a beautiful daily Bible verse on your church website. Includes widget, shortcodes, and prayer wall integration.

== Description ==

LordsGuide brings daily Scripture to your church website with a beautiful, customizable verse card that updates automatically every day.

**Features:**

* Daily Bible verse card with warm, elegant design
* WordPress widget for sidebar/footer placement
* Shortcodes for embedding anywhere: `[lordsguide_verse]`, `[lordsguide_prayer]`, `[lordsguide_app]`
* Responsive design that works on all devices
* No configuration needed — install and go
* Zero external dependencies — verses are built in
* Lightweight — no JavaScript libraries loaded

**For All Christian Traditions:**
Catholic, Protestant, Orthodox, Evangelical, Pentecostal — the daily verse speaks to all who follow Christ.

**Powered by LordsGuide** — AI-powered sermon prep, Bible study & faith community at [lordsguide.com](https://lordsguide.com)

== Installation ==

1. Upload the `lordsguide` folder to `/wp-content/plugins/`
2. Activate through the 'Plugins' menu
3. Add the "LordsGuide Daily Verse" widget to any sidebar
4. Or use `[lordsguide_verse]` shortcode in any post or page

== Shortcodes ==

* `[lordsguide_verse]` — Beautiful daily verse card
* `[lordsguide_prayer]` — Prayer wall link with CTA
* `[lordsguide_app height="600"]` — Full LordsGuide app embed

== Frequently Asked Questions ==

= Does this plugin require an account? =
No. The daily verse widget works without any account or API key.

= How often does the verse change? =
Every day at midnight, the verse rotates through a curated collection of 18 beloved Scripture passages.

= Is this for Catholics too? =
Yes! LordsGuide is designed for all Christians — Catholic, Protestant, Orthodox, and all traditions.
